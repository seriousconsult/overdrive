<h3 class="section" id="howto-disable-webrtc" name="howto-disable-webrtc"><a href="#howto-disable-webrtc" title="Click to expand">How to Disable WebRTC</a></h3>
<div class="none">
<dl>
  <dt><h4>WebRTC in Firefox</h4></dt>
  <dd>
    <p>To disable WebRTC in Firefox:</p>
    <li>Type <code>about:config</code> in the address bar and press Enter.</li>
    <li>In the search bar, type <code>media.peerconnection.enabled</code> and double-click the preference to set its value to <code>false</code>.</li>
  </dd>
</dl>
<dl>
  <dt><h4>WebRTC in Chrome</h4></dt>
  <dd>
    <p>The recommended solution to limit the risk of IP leakage via WebRTC is to use the official Google extension called <a href="https://chrome.google.com/webstore/detail/webrtc-network-limiter/npeicpdbkakmehahjeeohfdhnlpdklia" rel="noopener" target="_blank">WebRTC Network Limiter</a>, which provides a range of options with varying levels of protection.</p>
  </dd>
</dl>
